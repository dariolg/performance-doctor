== Description ==

Performance Doctor è uno strumento avanzato per l'analisi e l'ottimizzazione delle prestazioni del tuo sito WordPress. Monitora l'impatto di ogni plugin attivo, rileva conflitti tra script e hook, e suggerisce azioni correttive per migliorare la velocità e la stabilità del sito.
3.  Vai su 'Strumenti' -> 'Performance Doctor' per avviare la tua prima analisi.

== Frequently Asked Questions ==

= Il plugin rallenta il mio sito? =
No, Performance Doctor esegue l'analisi solo quando richiesto manualmente dalla dashboard admin. Non ha impatto sul frontend del sito a meno che non si attivino le ottimizzazioni specifiche.

= Posso annullare le modifiche? =
Sì, il plugin include un sistema di backup e rollback integrato. Puoi annullare qualsiasi ottimizzazione applicata con un semplice click.
